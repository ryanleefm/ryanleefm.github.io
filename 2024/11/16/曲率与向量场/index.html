曲率与向量场

数学
发布日期:   2024-11-16
  • 微分几何建立在微分流形上，但比微分流形多了测量长度的“尺”，即（黎曼）度量。
  • 黎曼流形：定义向量空间中的内积 -> 此内积可用矩阵表示 -> 黎曼度量即在流形切空间上赋予光滑内积 -> 通过类似单位分解可以证明每个流形都存在一个黎曼度量
  • 曲线：
  • 空间中的曲面：定义曲面上各种曲率 -> 叙述曲面上的高斯绝妙定理与GB定理
  • 欧氏空间中的方向导数：回顾定义，几何意义=函数或向量场在某方向上的变化率 >> 它满足的性质，即线性与Leibniz法则，以及零挠率、零曲率、与度量相容 >> 定义沿曲线、子流形的（光滑）向量场 >> 定义欧氏空间子流形上的方向导数，几何意义=在该子流形某切方向上，函数或向量场变化率 , 以及其与欧氏空间中方向导数相同的性质，注意不含零挠率。
  • 形状算子：定义， 几何意义：就是中曲面单位法向量沿切方向的方向导数（变化率） >> 它的几条运算性质 >> 借它定义第一、二基本形式，几何意义：两切向量的内积、一切方向上法向量变化率在另一切向量上的投影,即 >> 高斯曲率与形状算子跟第二基本形式的关系
  • 仿射联络： 根据线性和Leibniz法则定义， 几何意义：立足于流形切空间中看方向导数 >> 定义仿射联络的挠率与曲率，可证都是线性的 >>> 定义黎曼联络，为零挠率和与度量相容 >> 黎曼联络的唯一性 >> 例： 中曲面的黎曼度量
  • 向量丛：
  • 高斯绝妙定理： 几何意义：高斯曲率（可由此式算得）是一个等度变换下的不变量
  • 推广到中的超曲面： 再定义形状算子、黎曼联络等
